# SQLCipher CLI

此目录随仓库内置 Windows x64 UCRT 版 SQLCipher CLI，使下载源码 zip 或 `git clone` 后可以直接使用：

```powershell
python scripts\qqnt_windows_export.py --kill-qq-first --account YOUR_QQ_NUMBER --outdir RE\windows_qq_export --sqlcipher tools\sqlcipher\sqlcipher.exe
```

当前文件来自：

```text
https://github.com/ShintoKosei/sqlcipher-windows-builds/releases/latest
```

如需更新，重新下载最新 release zip 并解压覆盖本目录即可。
